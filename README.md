# 今日の言い訳メーカー 🐢

ボタンを押すだけで、今日の言い訳をランダム生成するアプリです。

---

## 機能

- カテゴリ選択（仕事 / 勉強 / 家事 / 運動）
- カテゴリに応じた言い訳をランダムで1件表示
- 生成した言い訳をワンクリックでコピー

## 技術スタック

- [Next.js](https://nextjs.org/) 16 (App Router)
- TypeScript
- Tailwind CSS v4
- React `useState` による状態管理

## 開発環境のセットアップ

```bash
pnpm install
pnpm dev
```

http://localhost:3000 で起動します。

## ビルド・起動

```bash
pnpm build
pnpm start
```
